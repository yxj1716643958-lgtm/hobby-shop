# UKA模工坊 - 部署到公网指南

## 方案1：Vercel 部署（推荐 - 免费且简单）

### 步骤：

1. **注册 Vercel 账号**
   - 访问：https://vercel.com
   - 使用 GitHub、GitLab 或邮箱注册

2. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

3. **登录 Vercel**
   ```bash
   vercel login
   ```

4. **部署网站**
   在 hobby-shop 目录下运行：
   ```bash
   vercel
   ```

   按照提示操作：
   - 选择项目设置
   - 确认部署

5. **完成！**
   - 部署完成后会得到一个公网地址，如：`https://uka-model-workshop.vercel.app`
   - 所有人都可以访问这个地址

### 优点：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 自动部署（推送代码后自动更新）
- ✅ 无需服务器维护

## 方案2：使用 GitHub + Vercel（更简单）

1. **将代码上传到 GitHub**
   ```bash
   cd hobby-shop
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/uka-model-workshop.git
   git push -u origin main
   ```

2. **在 Vercel 导入项目**
   - 登录 https://vercel.com
   - 点击 "New Project"
   - 选择你的 GitHub 仓库
   - 点击 "Deploy"

3. **完成！**
   - 每次推送代码到 GitHub，网站会自动更新

## 绑定自定义域名

部署后，你可以绑定自己的域名（如 ukamodel.com）：

1. 在域名注册商购买域名
2. 在 Vercel 项目设置中添加域名
3. 按照提示配置 DNS 记录

## 推荐流程（最快5分钟）

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 进入项目目录
cd hobby-shop

# 3. 登录并部署
vercel login
vercel

# 4. 生产环境部署
vercel --prod
```

完成后你会得到一个公网地址，所有人都可以访问！
