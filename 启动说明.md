# UKA模工坊 启动指南

## 快速启动

### 1. 安装依赖
打开命令提示符（CMD）或 PowerShell，进入项目目录：

```bash
cd hobby-shop
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问网站
在浏览器中打开：
```
http://localhost:3000
```

## 其他命令

### 构建生产版本
```bash
npm run build
```

### 启动生产服务器
```bash
npm start
```

### 代码检查
```bash
npm run lint
```

## 项目说明

这是一个基于 Next.js 14 构建的现代化电商网站，包含：

- ✅ 响应式设计（支持手机、平板、电脑）
- ✅ 顶部导航栏（搜索、用户、购物车）
- ✅ 轮播横幅
- ✅ 商品展示网格
- ✅ 页脚信息
- ✅ 中文界面

## 技术栈

- **框架**: Next.js 14 (React 18)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **开发环境**: Node.js

如果遇到问题，请检查：
- Node.js 是否已安装（建议 v18 或更高版本）
- npm 是否正常工作
- 端口 3000 是否被占用
