# 模型杂志资料库 启动指南

## 快速启动

### 1. 安装依赖
打开命令提示符（CMD）或 PowerShell，进入项目目录：

```bash
cd hobby-shop
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问网站
在浏览器中打开：
```
http://localhost:3000
```

## 其他命令

### 构建生产版本
```bash
npm run build
```

### 启动生产服务器
```bash
npm start
```

### 代码检查
```bash
npm run lint
```

## 项目说明

这是一个基于 Next.js 14 构建的模型杂志资料展示网站，包含：

- ✅ 响应式设计（支持手机、平板、电脑）
- ✅ 顶部导航栏（搜索、分类）
- ✅ 资料展示横幅
- ✅ 杂志资料网格展示
- ✅ 页脚信息
- ✅ 中文界面

## 资料说明

网站展示的杂志资料来自：`D:\模型杂志资料\【书的资料在这里】杂志宣发资料\封面 宣传`

包含12+本模型杂志，涵盖：
- 高达系列
- 假面骑士系列
- Ma.K系列
- 模工坊月刊
- 各类制作指南

## 技术栈

- **框架**: Next.js 14 (React 18)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **开发环境**: Node.js
