# MAPPA Style Website

这是一个模仿 MAPPA 动画工作室官网风格的网站项目。

## 特点

- 简洁现代的设计风格
- 响应式布局
- Next.js 14 + TypeScript
- Tailwind CSS

## 安装

```bash
npm install
```

## 运行

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 构建

```bash
npm run build
npm start
```

## 项目结构

- `/app` - Next.js 应用路由和页面
- `/components` - React 组件
- `/public` - 静态资源

## 自定义

你可以修改以下文件来自定义网站内容：

- `components/WorksSection.tsx` - 作品展示
- `components/NewsSection.tsx` - 新闻资讯
- `components/Hero.tsx` - 首页横幅
- `app/globals.css` - 全局样式
